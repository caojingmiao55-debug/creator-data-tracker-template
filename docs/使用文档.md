# 创作者数据追踪系统 - 使用文档

## 在线访问

**仪表盘地址**：部署后获得（Vercel / GitHub Pages）

部署完成后，团队成员直接点击链接即可查看数据。

---

## 功能介绍

### 数据总览
- 总粉丝数、总播放量、总点赞数、总收藏数、作品总数
- 支持按时间筛选：近7天 / 近30天 / 全部

### 各平台数据
- **抖音**：粉丝、播放量、点赞、收藏、作品数、曝光量
- **小红书**：粉丝、播放量、点赞、收藏、作品数、曝光量
- **视频号**：粉丝、播放量、点赞、收藏、作品数、曝光量

### 数据图表
- 粉丝增长趋势
- 各平台数据对比

### 热门内容
- TOP 5 热门作品排行（按播放量排序）

---

## 数据更新

### 自动更新
- **抖音、小红书**：每天早上 10:00 自动采集
- 采集后自动同步到线上仪表盘

### 手动更新
如需立即更新数据：
```bash
cd /path/to/creator-data-tracker
source .venv/bin/activate
python collect_all.py
```

### 视频号数据更新（⚠️ 特别说明）

> **视频号自动抓取功能待更新**
>

## 常见问题

### Q: 数据没有更新？
检查 Cookie 是否过期，参考《配置文档》更新 Cookie。

### Q: 如何监测其他账号？
更换 `config.json` 中对应平台的 Cookie 即可，详见《配置文档》。

### Q: 页面打不开？
检查网络连接，或访问 https://vercel.com/dashboard 查看部署状态。
